# Angular2 + Electron

**基于Angular2和Electron的项目框架**

**搭建一个基础框架，结构体系清晰，便于项目开发。**

## 环境

**Node ` v4.x.x` && npm ` 3.x.x`**

## 项目

**步骤**
```
git clone https://github.com/Angular2Electron/Angular2-Electron.git
cd Angular2-Electron
npm install
npm start
```
**npm script**
```
npm start 
npm run tsc 
npm run tsc:w
npm run lite
```
**npm test**
```
npm test
npm run e2e
```
